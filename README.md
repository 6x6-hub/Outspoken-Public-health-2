# OUTSPOKEN PUBLIC HEALTH

**AI-First Community Public Health Surveillance and Health Management Platform for Zambia**

## Overview

OUTSPOKEN PUBLIC HEALTH is an intelligent digital health platform that bridges the gap between community members, Community Health Workers, clinics, hospitals, and health authorities across Zambia.

Powered by **OUTSPOKEN AI ASSISTANT**, the platform provides:

- 🤖 **Conversational AI Health Guidance** – Real-time streaming responses with multilingual support
- 📸 **Smart Image & Document Analysis** – Upload photos, medical reports, lab results
- 🗣️ **Voice-First Interface** – Speech-to-text and text-to-speech for accessibility
- 📊 **Disease Surveillance** – Automated outbreak detection and trend analysis
- 👨‍⚕️ **Healthcare Professional Tools** – Clinical decision support and patient management
- 🌍 **Offline-First Design** – Works without internet, syncs when connected
- 🔒 **Enterprise Security** – End-to-end encryption, audit logs, role-based access

## Quick Start

```bash
# Clone repository
git clone https://github.com/6x6-hub/outspoken-Public-health-2.git
cd outspoken-Public-health-2

# Install dependencies
npm install

# Start development
npm run dev

# Or with Docker
npm run docker:up
```

The application will be available at:
- **Frontend:** http://localhost:3000
- **API:** http://localhost:3001

## Technology Stack

- **Frontend:** React + Tailwind CSS + Framer Motion
- **Backend:** Node.js + Express.js
- **Database:** PostgreSQL + Redis
- **AI/ML:** OpenAI GPT-4, Vision API
- **Authentication:** JWT + Biometric Support
- **Deployment:** Docker, Kubernetes
